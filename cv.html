James Waples

Experienced Rust developer

Edinburgh, Scotland

Hi, I'm James, a Rust, electronics and CNC enthusiast. I've been programming since I was around 10 years old, first starting with 8 bit assembler, Visual Basic and C. I started my career at 18 with a PHP web development role and slowly moved towards the JavaScript/Typescript ecosystem as it grew around the popularity of libraries like React. My work now revolves around the wonderful Rust programming language, gladly using it both professionally and in open source to write high quality code in embedded environments.

I'm motivated by interesting software problems, challenging optimisations, interesting performance debugging and making a polished product for people to use. I strongly believe we work best in a team, each of us adding to something greater than the sum of parts.

I'm also interested in electronics. As it's naturally close to programming, being the hardware one's code runs on, it was a quick discovery after computers and has become a long running hobby, most recently in Rust on STM32.

Melding electronics and programming lead me to discover the expensive yet fascinating world of DIY CNC machines, myself having built at least 5 of improving capabilities (and heft) over time.

Open source projects

  • https://github.com/ethercrab-rs/ethercrab

    EtherCrab is a pure-Rust EtherCAT master runtime born of the frustrations of working with existing C-based libraries. It's an interesting introduction to the world of industrial automation and German beaurocracy when applying for (and receiving) EtherCAT organisation membership.

  • https://github.com/jamwaffles/linuxcnc-hal-rs

    Rust bindings and a high level API to allow writing Rust modules for the popular open source LinuxCNC motion controller.

  • https://github.com/embedded-graphics/embedded-graphics

    A graphics library for embedded applications using no dynamic allocation, with primitives, fonts and image support. I'm proud to say it has over 400,000 downloads and a vibrant community around it.
